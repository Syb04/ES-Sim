# サブエージェント指示: フロントエンド 可視化強化

対象リポジトリ: `/home/claude/ES-Sim/frontend`(npm install 済み)

## 背景

ES-Sim は 2D静電場FEMアプリ(Tauri 2 + React 18 + TS strict)。`CadCanvas.tsx` が
解析結果(`SolveResult`: 節点電位 v、要素ごとの e_field)を電位カラーマップで描画している。
これに等電位線・|E|表示・ベクトル矢印・カラーバーを追加する。仕様書 §7 参照。
※ 直前に別エージェントがスケッチツールを実装しているので、現状のコードを必ず読んでから着手すること。

## やること

### 1. `src/canvas/isolines.ts` を新規作成

- `computeIsolines(nodes, triangles, v, levels): Point[][][]`
  各等値レベルについて、三角形ごとの線形補間でエッジ交点を求め、線分群を返す
  (マーチング・トライアングル。線分の連結・ポリライン化まではしなくて良い)
- レベルは v_min〜v_max を等分(本数は呼び出し側から指定、デフォルト15)

### 2. CadCanvas.tsx — 表示モードとオーバーレイ

props(App から)で制御:

- **fieldView**: `"v" | "e_abs"` — カラーマップの対象を切替。
  `e_abs` は要素ごとの |E| で三角形を塗る(要素値なので節点平均不要)。スケールは 0〜max の線形
- **showIsolines**: 等電位線オーバーレイ(細い黒線 `rgba(0,0,0,0.5)`)。fieldView とは独立に表示可
- **showVectors**: 要素重心に E 方向の矢印。画面上の矢印本数が最大 ~600 になるよう要素を間引く
  (要素数/600 の間隔でストライド)。長さは一定(例 14px)で向きのみ、色は白系
- **カラーバー**: キャンバス右下に縦グラデーションのカラーバー + min/max ラベルを canvas 直描き
  (fieldView に応じて単位 V / V/m を表示)

### 3. App.tsx — トグルUI

ツールバーまたはサイドパネルに fieldView 切替(V / |E|)、等電位線・ベクトルのチェックボックスを追加。
