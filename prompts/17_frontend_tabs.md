# サブエージェント指示: サイドパネルのタブ分け

対象リポジトリ: `/home/claude/ES-Sim/frontend`(npm install 済み)

ES-Sim(Tauri 2 + React 18 + TS strict)。サイドパネルの項目が縦に長くなりすぎたため、
タブで整理する。必ず現状の App.tsx / panels/ を読んでから着手すること。

## タブ構成

サイドパネル最上部に**保存/読込ボタンの行**(全タブ共通)、その下に**3つのタブ**:

1. **静電場**: ジオメトリ(domain 幅/高さ)、境界条件(4辺、RF含む)、メッシュ(サイズ)、
   領域一覧+選択中領域のプロパティ編集、解析結果サマリ(節点数・V範囲・エネルギー等)、
   メッシュ結果サマリ
2. **粒子追跡**: 粒子種、エミッタ、エネルギー分布、積分設定、Trace実行ボタンと結果サマリ
   (現 ParticlePanel の内容)
3. **PIC**: 初期プラズマ、注入、計算設定、実行制御(開始/停止・進捗・警告)、診断+履歴チャート
   (現 PicPanel の内容)

## 実装要件

- タブバーはパネル上部に固定(スクロールしても常に見える)。選択中タブを視覚的に明示
- タブ切替は表示の切替のみで、**各タブ内の編集状態・実行状態は保持**されること
  (PIC 実行中に静電場タブへ移っても実行・ライブ描画は継続。アンマウントで WS が切れたり
  チャート履歴が消えたりしないよう、非表示は CSS(display:none)で行うか state を App 側に保持)
- キャンバス上での操作との連動:
  - 領域を選択したら自動で「静電場」タブに切替(プロパティがすぐ見えるように)
  - エミッタツールで線を確定したら「粒子追跡」タブに切替
- 既存の機能・レイアウト(キャンバス、ツールバー、プロファイルパネル)は変更しない
- スタイルは既存のダークテーマ CSS 変数に合わせ、`src/style.css` にタブ用クラスを追加

## 制約

- 新しい npm 依存を追加しない。コメントは日本語。TS strict 維持
- コンポーネント分割は自由(例: `src/panels/FieldPanel.tsx` に静電場タブを切り出すなど)
