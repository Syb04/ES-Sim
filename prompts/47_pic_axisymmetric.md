# 47: PIC の軸対称 (rz / rz_x0) 対応 — 設計メモ

※ この機能は物理項目が主のため、サブエージェントではなく本体モデルが直接実装した。
本ファイルはコード中の `prompts/47` 参照に対応する設計記録である。

## 方針

マクロ粒子を「リング」(周方向に一様な実粒子 w 個の輪) として扱う標準的な 2d3v
軸対称 PIC。`v[:, 2]` は xy モードの vz の代わりに vθ (周方向速度) を表す。

## 規約 (fem.assemble との整合が要)

fem.assemble の軸対称弱形式は
`∫ ε ∇V·∇W r dr dz = ∫ ρ W r dr dz`(両辺の 2π を落とした規約)。
このため:

- **リング電荷の P1 射影**: 総電荷 Q [C] のリング (半径 r_p) は
  ρ = Q δ(z−z_p)δ(r−r_p)/(2π r_p) なので、右辺への寄与は
  `f_i = Q·L_i(x_p)/(2π)`。堆積ベクトルと誘電体表面電荷 q_surf は物理量 [C] の
  まま保持し、`_solve_phi` 内で 1/(2π) スケールして加える (1箇所に集約)
- **要素体積**: V_e = 2π r̄ A (装荷重み w0 = n·V/n_macro、サンプリング確率、
  場エネルギー ½ε|E|²V_e)
- **節点体積**: V_i = Σ_e 2π ∫L_i r dA = Σ_e 2π(A/12)(2r_i + r_j + r_k)
  (線形 r の厳密積分。fem 右辺と同じ公式)。密度・電離レート・位相分解密度の規格化

## プッシュ (particles.py の trace と同じ規約)

- 遠心力項 a_r += vθ²/r を現在位置で評価 (半陰的)
- 移動後に角運動量保存 vθ = L/r (L = r_n·vθ_{n-1/2}) で更新
- 軸交差 (r < 0): r・vr・vθ (= L の符号) を鏡映してから walk する
  (軸 r=0 は境界メッシュエッジだが、鏡映により粒子は吸収されない)
- 初期装荷・注入・FN 放出の半ステップ後退キックにも遠心力項を含める

## 単位の変化 (フロントのラベル切替)

- 注入電流 `current_a_per_m`・FN 電流: xy = [A/m] (奥行き1m) → rz = [A] (リング全電流)
- 表面電荷 surf_q: [C/m] → [C]、場のエネルギー fe: [J/m] → [J]

## xy 経路のビット不変

- `elem_vol` は xy では `area` と同一オブジェクト (演算列が変わらない)
- rz 固有処理はすべて `if self.rz:` ガードで分岐 (xy の式は文字通り不変)
- 全102テスト (既存98 + 新規4) パスで回帰なしを確認

## 既知の近似

- 初期装荷の要素内サンプリングは一様 (要素選択のみ 2πr̄A 比例)。要素内の
  r∝ 依存を r̄ で代表するため、軸近傍要素では僅かに軸寄りが過剰になる
  (要素サイズ ≪ r で厳密な一様密度へ収束)

## 検証 (backend/tests/test_pic_rz.py)

1. 中性静穏開始 (電子・イオン同位置) → φ ≈ 0 (堆積の厳密相殺)
2. 一様帯電円柱 (イオンリングを要素重心へ決定的配置、L=10R) の中央部電位が
   解析解 φ(r) = ρ(R²−r²)/(4ε0) と誤差 5% 未満 (2π 係数と体積規格化の検証)
3. 無場自由粒子 (vθ0 のみ) の r(t) = √(r0²+(vθ0 t)²)・L 保存・|v| 保存 (1% 未満)
4. 軸方向プラズマ振動: 冷たい電子の正弦モードドリフトで KE が 2×fpe (10% 以内)
