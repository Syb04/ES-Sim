# 56: DSMC の軸対称 (rz / rz_x0) 対応 — 設計メモ

※ 物理項目が主のため本体モデルが直接実装した。コード中の `prompts/56` 参照に
対応する設計記録。定式化の詳細は docs/DSMC.md §4a を参照。

## 方針

- 粒子はリング (周方向一様)、v[:, 2] は vθ (PIC の rz 対応 prompts/47 と同じ規約)
- **自由飛行は3D直線の厳密な子午面射影**: DSMC は移動中に力が働かないため、
  r' = √((r + v_r Δt)² + (v_θ Δt)²) と速度の子午面回転で厳密に扱える
  (PIC の半陰的遠心力積分と違い近似なし)。r' ≥ 0 が常に成り立つので
  対称軸は物理境界にならない
- 軸上の境界メッシュエッジは数値的なかすり対策としてユーザ指定によらず鏡面に強制
- walk・境界交点判定・壁再放出後の残余レグは (z, r) 平面の直線分近似
  (Δt が小さければ二次の誤差)
- セル体積 2πr̄A、流入エッジは輪帯面積 2πr̄L (sccm 配分も輪帯面積比例)。
  xy では vol = area の同一オブジェクト参照で既存経路は数値不変

## 検証 (backend/tests/test_dsmc.py)

1. 密閉円筒 (軸=下辺): n/T/p 保持 3% 以内 + 内外半分の密度一致 8% 以内
   (2πr̄A 体積規格化の検証)、粒子数保存 (軸で消えない)
2. 軸方向自由分子流 (左端面リザーバ→右真空、外周鏡面): 密度 = リザーバの 1/2、
   u_z = c̄/2 (半空間 Maxwell の解析値) 5% 以内、質量収支 7% 以内
