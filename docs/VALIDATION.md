## 4. PIC-MCC 統合: Turner ベンチマーク ケース1 (He CCP)

**基準文献**: M. M. Turner, A. Derzsi, Z. Donkó, D. Eremin, S. J. Kelly, T. Lafleur,
T. Mussenbrock, "Simulation benchmarks for low-pressure plasmas: Capacitive discharges",
*Physics of Plasmas* **20**, 013507 (2013). doi:10.1063/1.4775084 —
独立5コードの相互比較で確立された PIC-MCC の標準ベンチマーク。
基準データ・断面積は [lase-unb/ccp-benchmark](https://github.com/lase-unb/ccp-benchmark)
経由で取得 (論文電子付録由来。`backend/verification/data/turner/`、LXCat条件により git 管理外)。

### 設定 (ケース1)

He、ギャップ 6.7 cm、n_g = 9.64e20 m⁻³ (300 K)、V = 450 V @ 13.56 MHz、
初期プラズマ 2.56e14 m⁻³ (Te=30000 K, Ti=300 K)、Δx = L/128、dt = 1/(400f)、
512,000 ステップ (1280 RF周期)、最後の32周期で時間平均。
断面積: 電子 Biagi 7.1 (弾性MT・励起19.82/20.61 eV・電離24.59 eV)、
イオン Phelps (等方+後方散乱、重心系エネルギー参照)。電離余剰エネルギーは等分配。

ES-Sim 側は 2D FEM-PIC を幅 2 mm のストリップ+上下エッジ鏡面反射で1D相当として実行
(粒子速度は3成分 = 2d3v)。マクロ粒子 65,536×2種、実行時間 9,521 s (2 vCPU, numpy)。

### 結果

| 指標 | n_e | n_i |
|---|---|---|
| 中心密度 ES-Sim [m⁻³] | 1.335e14 | 1.377e14 |
| 中心密度 基準解 [m⁻³] | 1.363e14 | 1.405e14 |
| **中心密度 相対差** | **−2.06%** | **−1.96%** |
| ピーク密度 相対差 | −2.06% | −1.69% |
| **プロファイル相対 L2 偏差** | **1.85%** | **1.66%** |

比較プロット: `backend/verification/turner_case1.png`(全ギャップにわたり
バルク・シースとも基準プロファイルにほぼ重なる)。

### 評価

密度プロファイル全体が基準解と **2%以内** で一致。Turner 論文における独立コード間の
ばらつき (~1%) には僅かに及ばないが、以下のモデル差を考慮すれば妥当な水準:

- 非構造三角形メッシュの FEM (基準コードは1D構造格子の有限差分)
- 2Dストリップ+鏡面反射による1D模擬 (統計・幾何の差)
- 時間平均試料数・マクロ粒子統計

### 再現方法

```powershell
cd backend
python verification/turner_case1.py               # 本計算 (数時間)
python verification/turner_case1.py --steps 4000  # スモーク (数分)
```
