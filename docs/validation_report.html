4. PIC-MCC 統合: Turner ベンチマーク ケース1(He CCP)

基準文献: M. M. Turner ほか, "Simulation benchmarks for low-pressure plasmas: Capacitive discharges", Physics of Plasmas 20, 013507 (2013). doi:10.1063/1.4775084 — 独立5コードの相互比較で確立された PIC-MCC の標準ベンチマーク。基準データ・断面積は lase-unb/ccp-benchmark 経由で取得(論文電子付録由来)。

設定(ケース1)

ガス / ギャップHe 300 K, n_g = 9.64×10²⁰ m⁻³ / L = 6.7 cm
駆動V = 450 V @ 13.56 MHz(片側、対向GND)
初期プラズマ2.56×10¹⁴ m⁻³(Te = 30000 K, Ti = 300 K)
離散化Δx = L/128 (0.523 mm), dt = 1/(400f) = 1.844×10⁻¹⁰ s
実行512,000ステップ(1280 RF周期)、最終32周期で時間平均。実行時間 9,521 s(2 vCPU, numpy)
断面積電子: Biagi 7.1(弾性MT・励起19.82/20.61 eV・電離24.59 eV)/ イオン: Phelps(等方+後方散乱、重心系参照)
衝突則等方散乱、電離余剰エネルギー等分配(Turner規定)
ES-Sim側の構成2D FEM-PIC を幅2mmストリップ+上下鏡面反射で1D相当として実行(2d3v、マクロ粒子 65,536×2種)

結果

指標n_en_i
中心密度 ES-Sim [m⁻³]1.335×10¹⁴1.377×10¹⁴
中心密度 基準解 [m⁻³]1.363×10¹⁴1.405×10¹⁴
中心密度 相対差−2.06%−1.96%
ピーク密度 相対差−2.06%−1.69%
プロファイル相対L2偏差1.85%1.66%

評価

密度プロファイル全体が基準解と2%以内で一致。Turner論文における独立コード間の ばらつき(~1%)には僅かに及ばないが、次のモデル差を考慮すれば妥当な水準と判断する: 非構造三角形メッシュのFEM(基準コードは1D構造格子FD)、2Dストリップ+鏡面反射による1D模擬、 時間平均試料数・マクロ粒子統計。
