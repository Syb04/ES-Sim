# DSMC(定常ガス流れ)アルゴリズム解説

`backend/es_sim/dsmc.py` に実装した DSMC(Direct Simulation Monte Carlo)法の
アルゴリズムをまとめる。目的は、装置内の希薄〜遷移領域のガス流れを定常まで解き、
要素ごとの数密度 $n$・温度 $T$・流速 $\mathbf{u}$・圧力 $p = n k_B T$ を得て、
PIC-MCC の非一様背景ガス場(`mcc.use_dsmc_gas`)として供給することである。
経緯・設計判断の一次資料は `prompts/54_dsmc_gas_flow.md`、検証テストは
`backend/tests/test_dsmc.py` を参照。

## 1. 概要と適用範囲

DSMC はボルツマン方程式の確率的粒子解法で、シミュレーション粒子1個が実分子
$W$ 個(マクロ重み)を代表する。移動(自由分子飛行 + 境界相互作用)と
セル内2体衝突(NTC 法)を時間刻み $\Delta t$ ごとに分離して進める標準構成
(Bird 1994)に従う。

- 座標系: 平面2D(`coord="xy"`、奥行き 1 m 換算でセル体積 = 面積 × 1 m)と
  **軸対称**(`coord="rz"` / `"rz_x0"`、粒子はリング・セル体積 = $2\pi\bar{r}A$。§4a)
- セル: 静電場 FEM と同じ三角形メッシュの要素をそのまま衝突セルに使う
  (walk 探索・隣接表など `particles.py` のインフラを再利用)
- 速度は 3 成分(xy: $(v_x, v_y, v_z)$、rz: 面内2成分 + $v_\theta$)、
  単原子分子(内部エネルギーなし)
- 有効性の目安: セル寸法 ≲ 平均自由行程 λ、$\Delta t$ ≲ 平均衝突時間・
  セル通過時間。連続体極限(高圧)では計算コストが増すだけで原理的には正しい

## 2. 分子モデル: VHS(Variable Hard Sphere)

全断面積は相対速さ $c_r$ に依存する VHS モデル:

$$\sigma_T(c_r) = \frac{\pi d_{\mathrm{ref}}^2}{\Gamma(5/2-\omega)}
\left(\frac{2 k_B T_{\mathrm{ref}}}{\mu\, c_r^2}\right)^{\omega-1/2}$$

- $d_{\mathrm{ref}}$: 基準温度 $T_{\mathrm{ref}}$ での基準直径
- $\omega$: 粘性の温度指数($\mu_{\mathrm{visc}} \propto T^\omega$、剛体球 = 0.5)
- $\mu = m/2$: 同種2体の換算質量
- 既定値は Ar: $m$ = 39.948 amu、$d_{\mathrm{ref}}$ = 4.17×10⁻¹⁰ m、
  $\omega$ = 0.81、$T_{\mathrm{ref}}$ = 273 K

散乱は重心系で等方(cosχ 一様・方位角一様)、$|\mathbf{g}|$ を保存して

$$\mathbf{v}_1' = \mathbf{v}_{\mathrm{com}} + \tfrac{1}{2}|\mathbf{g}|\,\hat{\mathbf{d}},\qquad
\mathbf{v}_2' = \mathbf{v}_{\mathrm{com}} - \tfrac{1}{2}|\mathbf{g}|\,\hat{\mathbf{d}}$$

とする(運動量・エネルギーを対ごとに厳密保存)。

## 3. 衝突: NTC(No-Time-Counter)法

セル $c$(粒子数 $N_c$、体積 $V_c$)ごとに、1ステップの衝突候補対数を

$$N_{\mathrm{cand}} = \frac{1}{2}\, N_c (N_c - 1)\, W\,
\frac{(\sigma_T c_r)_{\max,c}\, \Delta t}{V_c}$$

で決める(端数はセルごとに次ステップへ持ち越し)。候補対はセル内から一様に
2個抽選し、確率 $\sigma_T(c_r) c_r / (\sigma_T c_r)_{\max,c}$ で採択する。
$(\sigma_T c_r)_{\max,c}$ は初期値($c_r = 2 v_{mp}$ で評価)から始めて
実測最大値へ随時更新する。

実装はセル横断で完全ベクトル化している(候補対を全セルまとめて生成し、
`argsort` によるセル内グルーピングでインデックスを引く)。

**重要な数値対策(重複対の排除)**: 候補対をベクトル一括更新すると、同一粒子が
同ステップ内で複数の対に選ばれた場合に「古い速度で計算した相手側の更新が
上書きされる」ためエネルギー保存が壊れ、分散が湧いて系が加熱する
(実測で数百ステップに温度が数倍化)。対策として各粒子について**最初に現れた
対のみを実行**し、落とした対はセルの候補数持ち越しに加算して次ステップで
再抽選する。修正後は衝突フェーズのエネルギー変化が丸め誤差(〜1e-16)のみに
なることを確認済み。

## 4. 移動と境界条件

各ステップ、粒子を $\mathbf{x} \leftarrow \mathbf{x} + \mathbf{v}_{xy}\Delta t$ で
直線移動し、walk 探索(重心座標の隣接要素歩き)で所属セルを追従する。
境界メッシュエッジに達した粒子は、境界エッジの重心座標 $L=0$ を線形補間した
**交点**で処理する(交点からエッジ高さ×10⁻³ だけ内側へ配置)。

境界の適用範囲は `dsmc.boundaries` で **domain 外周のエッジ番号**または
**外周上の線分 p1–p2(部分区間指定)**で与える(両方指定は和集合。
電極と外枠の隙間などエッジの一部だけを流入口にする用途は線分指定を使う)。
判定は境界メッシュエッジの中点が指定線分上にあるかで行う(エッジ番号は
厳しい許容 10⁻⁸、ユーザ入力の線分はグリッドスナップ誤差を見込んだ 10⁻⁶ 相対)。
未指定の境界と conductor 領域の輪郭は拡散反射壁になる。

境界タイプ:

- **wall(拡散反射壁)**: 完全適応。壁温 $T_w$ の流束 Maxwell で再放出する。
  法線成分は流束重み付き分布 $P(v_n) \propto v_n e^{-v_n^2/2\sigma^2}$ から
  $v_n = \sigma\sqrt{-2\ln U}$($\sigma = \sqrt{k_B T_w/m}$)、接線・$z$ 成分は
  $\mathcal{N}(0, \sigma)$
- **symmetry(鏡面反射)**: 面内法線速度成分を反転($|\mathbf{v}|$ 厳密保存)
- **inlet(圧力指定、pressure_pa)/ outlet(圧力リザーバ)**: 到達粒子は吸収し、
  リザーバ側からの平衡流入(§5)で置き換える。圧力・温度の平衡だめとして振る舞う
- **inlet(流量指定、flow_sccm)**: 指定流量(§5)を注入し、入射してくる粒子は
  拡散反射壁として扱う(吸収しない)。これにより**正味流量が指定値に厳密に
  一致**する(圧力リザーバと違い、境界前の圧力は流量と下流コンダクタンスから
  自己無撞着に決まる)
- **outlet(真空排気、圧力省略/0)**: 吸収のみ

**重要な数値対策(残余移動時間の完走)**: 壁・対称で再放出した粒子は、
残余時間 $(1-\mathrm{frac})\Delta t$ 分の移動を続行し、再度境界に達したら同じ
処理を繰り返す(最大6レグ)。残余時間を捨てると「放出直後(平均エネルギー
$2k_BT_w$ の流束 Maxwell)の粒子が壁近傍に滞留する」バイアスが生じ、
気体温度が定常で +2% 程度高く出る(平衡箱テストで実測)。完走処理により
平衡箱の温度保持は 0.5% 以内に収まる。

**固体(誘電体)要素**: メッシュ化はされているがガスは侵入できない。進入した
粒子は移動を取り消し、壁温の等方 Maxwell で速度を再抽選する(質量保存を守る
簡易処理)。

## 4a. 軸対称(rz / rz_x0)の扱い

粒子をリング(周方向一様)として扱い、力が働かない自由飛行には
**3D 直線飛行の厳密な子午面射影**を使う:

$$r' = \sqrt{(r + v_r \Delta t)^2 + (v_\theta \Delta t)^2},\qquad
v_r' = \frac{v_r (r + v_r\Delta t) + v_\theta (v_\theta \Delta t)}{r'},\qquad
v_\theta' = \frac{v_\theta (r + v_r\Delta t) - v_r (v_\theta \Delta t)}{r'}$$

($z$ 方向は $z' = z + v_z \Delta t$)。$r' \geq 0$ が常に成り立つため対称軸は
物理境界にならない(軸上の境界メッシュエッジは数値的なかすり対策として
鏡面に強制する)。walk・境界交点の判定は $(z, r)$ 平面の直線分近似で行う
($\Delta t$ が小さければ誤差は二次)。壁再放出後の残余レグも直線分近似。

- セル体積: $V_c = 2\pi \bar{r} A$(NTC の候補数・密度規格化・初期装荷の
  重み付けすべてに使用)
- 流入: エッジの輪帯面積 $A_e = 2\pi \bar{r}_e L_e$ に流束を掛ける
  (sccm 配分も輪帯面積比例)。sccm は軸対称では実流量そのもの(換算不要)
- 検証: 密閉円筒で n/T/p 保持(3%以内)+ 径方向一様性(体積規格化の検証、
  8%以内)、軸方向自由分子流でリザーバの 1/2 密度・$\bar{c}/2$ 流速(5%以内)
- 既知の近似: 初期装荷・流入位置のセル/エッジ内サンプリングは $\bar{r}$ 代表
  (要素サイズ ≪ r で厳密へ収束)。一様マクロ重みのため軸近傍セルの統計は粗い

## 5. 流入(圧力リザーバ / 流量指定)

流入エッジごとに実分子数流量 $\dot{N}_e$ [1/s] を前計算し、毎ステップ
$\dot{N}_e \Delta t / W$ 個(端数はエッジごとに持ち越し)を注入する。
位置はエッジ上一様、法線速度は流束重み付き Maxwell、接線・$z$ は熱速度の
正規分布。

**圧力リザーバ**(pressure_pa): 平衡気体の壁面流束

$$\dot{N}_e = \Phi L_e,\qquad
\Phi = \frac{1}{4} n_r \bar{c},\qquad
n_r = \frac{p_r}{k_B T_r},\qquad
\bar{c} = \sqrt{\frac{8 k_B T_r}{\pi m}}$$

流出吸収(§4)と組で「圧力境界」になる。

**流量指定**(flow_sccm): sccm(標準状態 273.15 K・101325 Pa の cm³/min)を

$$\dot{N} = Q_{\mathrm{sccm}} \cdot \frac{p_0}{k_B T_0} \cdot \frac{10^{-6}}{60}
\approx Q_{\mathrm{sccm}} \times 4.478\times10^{17}\ \mathrm{1/s}$$

で分子数流量に換算し、指定範囲の各メッシュエッジへ長さ比例で配分する
($\dot{N}_e = \dot{N}\, L_e / \sum L$)。2D なので奥行き 1 m 換算である
(実装置のスロット長 $D$ [m] の流量を模すなら $Q/D$ を与える)。
注入は端数持ち越しの決定的処理なので、時間平均流量は換算値に厳密に一致する
(検証テストでは 10 sccm 指定で流入レート一致 2% 以内・定常質量収支 10% 以内)。

## 6. マクロ重み・時間刻みの決め方

- マクロ重み: $W = n_{\mathrm{ref}}\, A_{\mathrm{gas}} / N_{\mathrm{target}}$。
  $n_{\mathrm{ref}}$ は初期充填密度と全リザーバ密度の最大値(定常粒子数が
  目標 `n_particles` の近傍になる)
- $\Delta t$(自動): $0.25\, h_{\min} / v_{mp}$。$h_{\min}$ は最小要素の平均辺長、
  $v_{mp} = \sqrt{2 k_B T_{\mathrm{hot}}/m}$ は系内最高温度での最確速さ
  (セル通過を1ステップ1セル程度に抑える)

## 7. サンプリング(定常時間平均)

最終 `avg_steps` ステップでセルごとに粒子数・$\sum\mathbf{v}$・$\sum|\mathbf{v}|^2$
を積算し、

$$n = \frac{\langle N_c \rangle W}{V_c},\qquad
\mathbf{u} = \langle \mathbf{v} \rangle,\qquad
T = \frac{m}{3 k_B}\left(\langle |\mathbf{v}|^2 \rangle - |\langle \mathbf{v} \rangle|^2\right),\qquad
p = n k_B T$$

を得る。定常判定は自動化しておらず `n_steps` に委ねる(目安: 音速・熱速度での
系通過時間の数倍 + 衝突緩和)。平均区間の流入・流出実分子数も集計し、
定常の質量収支確認に使える。

## 8. PIC-MCC への結合(一方向)

`POST /dsmc` の結果はサーバー内に保持され、PIC 実行時に
`pic.mcc.use_dsmc_gas: true` で `GasField {n_g, T_g, u_g}`(要素ごと)として
MCC に注入される。MCC 側の扱い:

- null-collision の $\nu_{\max}$ は**最大密度** $n_{\max}$ で評価し、粒子ごとの
  プロセス採択時に局所密度比 $n_g(\mathbf{x})/n_{\max}$ を掛ける(非一様密度でも
  null-collision 法は厳密なまま)
- イオン衝突・電離生成イオンの背景 Maxwell は局所温度 $T_g(\mathbf{x})$ を使い、
  局所流速 $\mathbf{u}_g(\mathbf{x})$ をドリフトとして加える(CX の「中性風」効果)
- 定数場は一様指定とビット単位で一致(実装検証済み)

メッシュ(要素数)が DSMC 実行時と異なる場合は PIC 起動時にエラーになる
(ジオメトリやメッシュ設定を変えたら DSMC を再実行する)。

プラズマからガスへの反作用(ガス加熱・電離減耗・運動量移行)は扱わない
一方向結合である。必要になれば準定常反復(PIC → ソース項 → DSMC 再計算)が
将来課題。

## 10. 既知の制約・注意

- 単原子分子のみ(内部エネルギー・回転/振動緩和なし。Ar・He 等の希ガス向け)
- 単一ガス種(混合気体は未対応)
- 壁の適応係数は 1 固定(完全拡散反射)。鏡面成分の混合は未対応
- 固体(誘電体)要素の簡易処理(§4)は表面を厳密な壁として扱わない近似
- 同一ステップ内で複数対に選ばれた候補の繰り延べ(§3)は衝突頻度をわずかに
  遅らせるが、持ち越しにより時間平均のレートは保存される
- 統計ノイズはセルあたり粒子数 $\propto$ `n_particles` で決まる。目安として
  1セルあたり 20 個以上(既定 50,000 粒子は数千セルのメッシュに対応)
