# 同軸円筒メッシュ収束検証結果

仕様書 §6 の検証ケース「同軸円筒(2D断面): V(r) の対数分布」の収束確認結果。

- 内導体半径 a = 0.01 m (V1 = 100.0 V)
- 外導体半径 b = 0.04 m (0 V)
- 解析解 V(r) = V1 * ln(b/r) / ln(b/a)
- 解析解 C = 2 pi eps0 / ln(b/a) = 4.013037e-11 F/m
- domain外周・内導体は多角形近似(1辺の長さ ≒ mesh size / 4 となるよう分割数を各メッシュサイズごとに自動決定。gmshの挙動上、多角形分割数を固定すると mesh size の変更がメッシュ解像度に反映されないため、分割数を可変にしている)
